# My Website — Digital Marketing Internship Project

A simple website built for a digital marketing internship, including a content calendar for an e-commerce brand.

## Project Structure

```
my-website/
│── index.html       (main webpage)
│── style.css        (all styles)
│── script.js        (interactivity)
│── images/          (place your images here)
│── README.md        (this file)
```

## How to Run Locally

Just open `index.html` in your browser — no setup needed.

## How to Host on GitHub Pages

1. Upload this folder to a new GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your site will be live at: `https://yourusername.github.io/my-website/`

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
